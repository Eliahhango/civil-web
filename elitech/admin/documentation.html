Getting Started

The EliTechWiz CMS allows you to easily edit site text, perform global word replacements, and manage SEO metadata. Sign in via your administrator email to access the dashboard tools.

Text Replacements

Use the "Text Replacements" tab to globally find and replace strings across the entire site without writing code, allowing for rapid iteration on copy.

Custom Rules

Custom rules allow you to target specific paths (like `/elitech/projects/*`) to apply unique JSON configurations or targeted text overrides before deployment.
