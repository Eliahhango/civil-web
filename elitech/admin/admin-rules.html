Admin Operations Rules

Operational standard operating procedures for the system.

Content Management Standards

  • Quality Assurance: All text replacements must be visually inspected using the 'Preview' function prior to server deployment. Broken tags or bad JSON structures will block deployment.
  • Brand Voice: Ensure all modified content on the civil portal maintains the professional, engineering-focused tone of EliTechWiz.
  • SEO Compliance: Do not remove or blank out metadata tags (Title, Description). If modifying, ensure they remain compliant with current marketing initiatives.

Security Protocols

  • Authentication: Never share email links or passwords. Sessions are uniquely tied to the issuing IP address when possible.
  • Data Export: When downloading `content.json` backups, store the files in the authenticated company Drive only.
